# Running Backend
`cd backend`<br>
`npm i`<br>
`node index.js`

# Running Frontend
`cd frontend`<br>
`npm i`<br>
`npm run dev`

# Note
Default Frontend Url and Backend Url is set. If they are different/changed, environment variable needs to be updated.
